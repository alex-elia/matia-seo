Plateforme Agent SEO / GEO

Les yeux sur votre visibilité search

Matia est un agent IA qui définit votre stratégie de visibilité — objectifs, ICP, intentions — propose des actions, et exécute le travail technique et éditorial après votre approbation.

La plupart des outils SEO génèrent des balises. Matia observe, planifie et déploie.

Voir sur GitHub Démarrer

Pourquoi Matia

Du grec μάτια (yeux). Le SEO et le GEO, c'est être vu — par Google, Bing, et les systèmes de découverte IA. Matia est votre vigie : elle détecte les lacunes d'inventaire, les problèmes d'indexation et les intentions manquantes, puis vous aide à agir avec rigueur.

Fonctionnement

  1. Stratégie — L'agent rédige objectifs, ICP, positionnement et carte d'intentions par site. Vous validez.
  2. Plan d'actions — Analyse des écarts vs pages existantes. Actions typées : nouvelles pages, métadonnées, surfaces GEO, liens, indexation.
  3. Approbation — Rien ne s'exécute sans votre accord.
  4. Exécution — Outils et agents déploient : métadonnées, sitemaps, llms.txt, facts.json, contenu, workflows Search Console.
  5. Bilan — Mesure de couverture et itération de la boucle stratégique.

Démarrage rapide

git clone https://github.com/alex-elia/matia-seo.git
cd matia-seo
npm install
npm run build
npx matia help

Structure dans l'app hôte :

src/seo/
  strategy.yaml   # objectifs & ICP approuvés
  registry.ts     # inventaire pages
  policies.ts     # règles d'indexabilité
  entity-maps.ts  # facts GEO
