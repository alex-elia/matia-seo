Why Matia

From Greek μάτια (eyes). SEO and GEO are about being seen — by Google, Bing, and AI-mediated discovery. Matia is your visibility watch: it sees inventory gaps, indexing issues, and missing intents, then helps you act with discipline.

How it works

  1. Strategy — Agent drafts goals, ICP, positioning, and an intent map per website. You approve.
  2. Action plan — Gap analysis vs current pages. Typed actions: new pages, metadata, GEO surfaces, links, indexing.
  3. Approve — Nothing runs without your sign-off.
  4. Execute — Package tools and agents ship changes: metadata, sitemaps, llms.txt, facts.json, content, Search Console workflows.
  5. Review — Measure coverage and iterate the strategy loop.

Quick start

git clone https://github.com/alex-elia/matia-seo.git
cd matia-seo
npm install
npm run build
npx matia help

Host app layout:

src/seo/
  strategy.yaml   # approved goals & ICP
  registry.ts     # page inventory
  policies.ts     # indexability rules
  entity-maps.ts  # GEO facts
